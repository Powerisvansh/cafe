## ✨ Features

| Feature | Description |
|---|---|
| 🎠 **3D Card Carousel** | Smooth 3D rotating menu cards with flip-on-hover effect |
| 🌑 **Dark Luxury Theme** | Premium dark background with gold accents |
| 🛒 **Live Order Cart** | Add items, track total, and checkout in real time |
| 📅 **Table Reservation Modal** | Animated modal with form validation |
| 💫 **Scroll Animations** | Elements reveal smoothly as you scroll |
| 🔢 **Animated Counters** | Stats count up when they enter the viewport |
| ✉️ **Contact Form** | Working contact form with toast notifications |
| 🌟 **Testimonials Slider** | Auto-scrolling infinite testimonials strip |
| 🎆 **Floating Particles** | Ambient gold particles in the hero section |
| 📱 **Fully Responsive** | Works beautifully on mobile, tablet & desktop |
| ⌨️ **Keyboard Navigation** | Arrow keys to navigate the carousel |
| 👆 **Touch / Swipe Support** | Swipe carousel on mobile devices |

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/Powerisvansh/cafe.git
cd cafe
```

### 2. Open in browser

```bash
# Simply open index.html in your browser
start index.html        # Windows
open index.html         # macOS
xdg-open index.html     # Linux
```

> No build tools, no dependencies, no npm install — just open and run! ✅

## 🎨 Design System

| Token | Value | Usage |
|---|---|---|
| `--gold` | `#c9a84c` | Accents, buttons, highlights |
| `--dark` | `#0d0d0d` | Primary background |
| `--dark2` | `#1a1a1a` | Section backgrounds |
| `--dark3` | `#242424` | Cards, panels |
| `--text` | `#e8e0d5` | Body text |
| `--muted` | `#9a8a7a` | Subtle text |

**Fonts used:**
- `Playfair Display` — Headings & brand (Google Fonts)
- `Inter` — Body text & UI (Google Fonts)

## 🌐 Deploy on GitHub Pages

1. Go to your repo on GitHub
2. Click **Settings** → **Pages**
3. Under **Source**, select `main` branch → `/ (root)`
4. Click **Save**
5. Your site will be live at:

```
https://powerisvansh.github.io/cafe/
```
